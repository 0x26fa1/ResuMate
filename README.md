
# ResuMate

AI-powered resume builder and analyzer that helps students and job seekers generate ATS-friendly resumes, get instant feedback, and tailor content to target roles. Upload a resume, receive structured suggestions and summaries, then export a clean, professional version for applications.


> Live: [resumate.ph](https://resumate.ph/)
> 
> App: [app.resumate.ph](https://app.resumate.ph/)

## Features

- **Resume upload & storage** (Supabase Storage)
- **AI resume analysis** (Gemini API)
- **Structured feedback** (summary + actionable suggestions)
- **User authentication** (Supabase Auth)
- **Saved analysis results** in PostgreSQL
- **Modern UI** with Tailwind CSS


## Tech Stack

- **Frontend:** Next.js (App Router), React, TypeScript, Tailwind CSS  
- **Backend/API:** Next.js API Routes  
- **Database/Auth:** Supabase (PostgreSQL + Auth)  
- **File Storage:** Supabase Storage  
- **AI:** Google Gemini API
